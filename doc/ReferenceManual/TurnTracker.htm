Turn Counter

A Turn counter allows the current turn and phase of a game to be tracked. The current turn can be displayed in a small floating window, or docked into the toolbar

Name:  An identifying name for this Turn Counter.
Button text:  The text of the button to be added to the toolbar. 
Tooltip text:  The tooltip text of the button to be added to the toolbar. 
Hotkey:  Keyboard shortcut for the toolbar button.
Button Icon:  Icon for the toolbar button.
