Play Sound Button

The Play Sound Button plays the specified sound clip when it is pressed.

Name:  An identifying name for this button.
Button text:  The text of the button to be added to the toolbar. 
Tooltip text:  The tooltip text of the button to be added to the toolbar. 
Hotkey:  Keyboard shortcut for the toolbar button.
Button Icon:  Icon for the toolbar button.
Sound Clip:  The Sound clip file to be played.
Report Format:  A Message Format to report to the chat line when the sound plays.
