Module Installers

This tool allows you to create a downloadable standalone installer for your module. It creates a .jar file that you can make available for players to download. Double-clicking on the installer file will download and install files to the user's local machine, creating an executable that will load this module into VASSAL directly. The executable will use the same version of VASSAL as you are running when you create the installer, and the memory allocation you specify.
